User Stories for Time Tracker App
Registration: As a new user, I want to create an account with a username, email, and password so that I can have a private space for my data.

Secure Login: As a returning user, I want to log in with my email and password so that I can access my saved time entries safely.

Dashboard Overview: As a user, I want to see a clean home screen with a professional logo so that I can easily navigate the app's main features.

Track Time: As a user, I want to create new time entries for specific projects so that I can keep an accurate record of my work hours.

View Details: As a user, I want to click on a time entry to see its specific details and duration so that I can review my productivity.

Data Persistence: As a user, I want my data to be saved in local storage so that my entries are still there when I close and reopen the app.

External Sync: As a user, I want to see data fetched from an external API so that I can compare my tasks with global industry standards.

App Customization: As a user, I want to access a settings menu to toggle app preferences like notifications and theme.

Reminders: As a user, I want to receive a test notification alert so that I can verify the app is able to send me important reminders.
